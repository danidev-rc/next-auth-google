## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### Create a .env file with the following content:

```
# Google provider =>
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

#
NEXTAUTH_URL= # http://localhost:3000
NEXTAUTH_SECRET=
```
